# home-global-archive

本仓库用于归档和发布多个独立 HTML 页面。  
这些页面内容相互独立，不归属于任何特定域名或网站，仅作为静态文件存档使用。

## 项目简介

`home-global-archive` 是一个集中存放 HTML 页面资源的仓库。  
每个页面可以独立访问，内容涵盖不同主题，所有页面均以静态 HTML 形式保存。

本仓库的主要目的是为各类 HTML 页面提供一个长期、稳定的归档位置，方便检索与引用。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 存放独立 HTML 页面的目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 公共资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：每个 HTML 文件代表一个独立的归档页面。
- `assets/`：存放页面间共享的静态资源文件。

## 页面归档说明

- 每个页面均为完整的 HTML 文档，可直接在浏览器中打开。
- 页面之间无依赖关系，可单独访问。
- 归档内容不保证实时更新，但会定期整理与补充。
- 所有页面均保留原始结构与样式，仅作必要的基础适配。

## 维护说明

- 本仓库由维护者不定期更新。
- 如需提交新的归档页面，请将文件放入 `pages/` 目录并提交 Pull Request。
- 提交时建议附上简要说明，以便审核。
- 资源文件请统一放入 `assets/` 目录，避免散落。

## 许可

本仓库内容仅供存档与参考使用。  
如涉及第三方资源，请遵守相应版权规定。
